# Agent Docs

A personal collection of reusable AI-agent instructions, coding standards, workflow guardrails, documentation templates, and project handoff patterns. If it helps you, awesome! Free for everyone, always, foreva.

This repo is intentionally project-agnostic.

It is meant to help quickly set up consistent behavior for AI collaborators across game projects, software projects, and documentation work.

## Main Sections

- `agents/` — role/personality instructions for different AI collaborators.
- `standards/` — coding, documentation, changelog, and Git workflow rules.
- `templates/` — reusable starter files for new projects.
- `prompts/` — reusable task prompts for implementation, review, refactors, and docs.
- `archive/` — older versions kept for reference.

## Core Rule

Project-specific lore, mechanics, source files, and design bibles do not belong here.

This repo stores reusable operating rules.
